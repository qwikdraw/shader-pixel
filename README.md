# Shader Pixel
Integrating objects rendered in the fragment shader with a polygon scene
In C++ and Opengl

## Features
* Occulsion culling for shader objects
* Post processing with depth mapping
* Shared lighting between scene and shader objects

## Shader Objects

### Mandlebulb
### Mandlebox
### KIFS

### Dependencies
